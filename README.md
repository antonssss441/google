# Интеграция Google диска

Данный проект представляет собой набор Python-скриптов для взаимодействия с облачным хранилищем Google диск. Программа позволяет управлять файлами и папками через командную строку, используя официальное Google Drive API v3.

## Как установить

Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

`pip install -r requirements.txt`

## Как запустить проект

Для запуска введите в командной строке одну из четырех программ:

1. `python create_folder --name [Введите название папки] --id [Id папки]`
2. `python download_files --amount [количество файлов]`
3. `python upload_files --amount [количество файлов]`
4. `pethon get_sharing_link --amount [количество файлов]`
